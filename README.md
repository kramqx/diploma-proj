# Doxynix

Full-stack web application built as a doxynix project.

## Tech Stack

Frontend:
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS + shadcn/ui
- TanStack Query + Zustand
- Zod

Backend:
- tRPC
- Prisma + PostgreSQL (Neon)
- NextAuth (GitHub OAuth)
- Upstash QStash

Infrastructure:
- pnpm (monorepo)
- Render
- GitHub Actions (CI)
- Trello + GitFlow

## Getting Started

```bash
pnpm install
pnpm dev
